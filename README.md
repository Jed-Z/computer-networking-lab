# 计算机网络实验

*   中山大学（SYSU）
*   数据科学与计算机学院（SDCS）
*   计算机网络实验（Computer Networks Laboratory, DCS222）；教师：张永民
*   2019 春季学期（Spring 2019）



本仓库包括以下内容：

1.  计算机网络实验课程的全部内容，包括个人独立完成的**编程实验**和 4 人小组完成的**配置实验**。仓库中的代码和文档仅为我和我的小组完成的，无法保证正确性，仅供参考；
2.  小溪网上的理论练习题及参考答案，部分题目有解析。
